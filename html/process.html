process.process.process_ary_add(np_ary)¶

np_ary (np.array): e.g. np.ones((2,3))

Return:

number (int): e.g. 7

Example:
>>> import numpy as np
>>> np_ary = np.ones((2, 3))
>>> print(process_ary_add(np_ary)) # 7
